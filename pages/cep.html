💡 Como o POO foi usado aqui?

Criamos uma classe BuscaCEP para organizar toda a lógica de busca. Ela guarda o histórico de pesquisas como atributo e tem métodos para buscar, formatar e exibir os resultados.

// Classe que representa uma busca de CEP class BuscaCEP { constructor() { this.historico = []; // guarda as buscas anteriores } async buscar(cep) { const url = `https://viacep.com.br/ws/${cep}/json/`; const resposta = await fetch(url); // chama a API const dados = await resposta.json(); // converte pra objeto this.historico.push(cep); // registra no histórico return dados; } }
