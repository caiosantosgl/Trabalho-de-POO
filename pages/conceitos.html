Os fundamentos

Programação Orientada a Objetos

Os pilares que estruturam qualquer programa orientado a objetos.

01 O molde

Classe

Define a estrutura e o comportamento que os objetos terão. É como uma planta arquitetônica — descreve o que será criado, mas não é o objeto em si.

class Carro { // atributos da classe constructor(marca, ano) { this.marca = marca; this.ano = ano; } }
02 A ação

Método

Uma função definida dentro de uma classe. Representa os comportamentos e operações que os objetos daquela classe podem executar.

class Carro { acelerar() { // isso é um método! console.log("Acelerando!"); } }
03 O objeto

Instância

Um objeto criado a partir de uma classe. Cada instância tem seus próprios valores para os atributos, mas compartilha os métodos da classe.

// "new" cria uma instância (objeto) const meuCarro = new Carro("Honda", 2024); meuCarro.acelerar(); // chama o método
04 A propriedade

Atributo

São as variáveis que pertencem a um objeto. Guardam as características de cada instância. Dois carros podem ter marcas diferentes — cada um tem seu próprio atributo.

const carro1 = new Carro("Ford", 2020); const carro2 = new Carro("BMW", 2023); console.log(carro1.marca); // "Ford" console.log(carro2.marca); // "BMW"
05 A família

Herança

Uma classe pode herdar atributos e métodos de outra. A classe filha recebe tudo da classe pai e ainda pode ter suas próprias características.

class CarroEletrico extends Carro { carregar() { console.log("Carregando bateria!"); } } // herda marca e ano do Carro pai
06 A proteção

Encapsulamento

Esconder detalhes internos de um objeto, expondo apenas o necessário. Em JavaScript, usamos a convenção de underscore (_atributo) para indicar que algo é privado.

class ContaBancaria { constructor(saldo) { this._saldo = saldo; // "privado" } getSaldo() { return this._saldo; // acesso controlado } }
EXEMPLO COMPLETO — JavaScript
// 1. Definindo a CLASSE (o molde) class Animal { constructor(nome, especie) { this.nome = nome; // ATRIBUTO nome this.especie = especie; // ATRIBUTO especie } // MÉTODO — comportamento do objeto apresentar() { console.log(`Eu sou ${this.nome}, um ${this.especie}!`); } } // 2. HERANÇA — Cachorro é um tipo de Animal class Cachorro extends Animal { latir() { console.log(`${this.nome} diz: Au au!`); } } // 3. Criando INSTÂNCIAS (objetos reais) const gato = new Animal("Mimi", "gato"); const dog = new Cachorro("Rex", "cachorro"); // 4. Usando os MÉTODOS gato.apresentar(); // "Eu sou Mimi, um gato!" dog.apresentar(); // herdado de Animal dog.latir(); // próprio de Cachorro
